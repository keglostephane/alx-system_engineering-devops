# Some Shell scripts

## 0-current_working_directory
Print the absolute path name of the current working directory.

## 1-listit
Display the content list of your working directory.

## 2-bring_me_home
Change the working directory to the user's home directory

## 3-listfiles
Display directory contents in a long format

## 4-listmorefiles
Display current directory contents, including hidden files in long format

## 5-listfilesdigitonly
Display current directory contents
* Long format
* With users and group IDs displayed numerically
* And hidden files
